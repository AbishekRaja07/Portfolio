# Abishek R Portfolio

A modern, mobile-first portfolio website built with HTML, CSS, and vanilla JavaScript.

## Structure

- index.html
- style.css
- script.js
- assets/
  - images/
  - icons/
  - Abishek_R_Resume.pdf

## Customize

1. Replace `assets/Abishek_R_Resume.pdf` with your resume file.
2. Update project links and live demo URLs in `index.html`.
3. Replace placeholder images inside `assets/images/` if needed.

## Deploy (GitHub Pages)

1. Push this folder to a GitHub repository.
2. In GitHub, go to Settings → Pages.
3. Select `main` branch and root folder.
4. Your site will be live at `https://<username>.github.io/<repo>/`.
